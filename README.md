# AI Virtual Mouse using OpenCV & MediaPipe

An AI-powered virtual mouse system that uses hand gestures to control the computer cursor in real time.

## Features
- Hand tracking
- Cursor movement
- Real-time webcam processing
- OpenCV + MediaPipe integration

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## Run Project

```bash
pip install -r requirements.txt
python main.py

```
